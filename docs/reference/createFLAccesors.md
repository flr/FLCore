# Create accesor methods for a given class

<div class="ref-description section level2">

This function creates a complete set of standard S4 class accessors and
replacers. Not intended for direct use.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
createFLAccesors(class, exclude = character(1), include = missing)
```

</div>

</div>

<div class="section level2">

## Arguments

-   class:

    name of the class

-   exclude:

    Slot names to exclude

-   include:

    Slot names to include

</div>
