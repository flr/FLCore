# Class FLQuantPoint

<div class="ref-description section level2">

The `FLQuantPoint` class summarizes the contents of an `FLQuant` object
with multiple iterations along its sixth dimension using a number of
descriptive statistics.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
FLQuantPoint(object, ...)

# S4 method for class 'missing'
FLQuantPoint(..., units = "NA", n = 1)

# S4 method for class 'FLQuant'
FLQuantPoint(object, ..., probs = c(0.25, 0.75))

# S4 method for class 'FLQuantPoint'
n(object, ...)
```

</div>

</div>

<div class="section level2">

## Arguments

-   object:

    Input numeric object

-   ...:

    Additonal arguments

</div>

<div class="section level2">

## Details

An object of this class has a set structure along its sixth dimension
(*iter*), which will always be of length 5, and with dimnames *mean*,
*median*, *var*, *uppq* and *lowq*. They refer, respectively, to the
sample mean, sample median, variance, and lower (0.25) and upper (0.75)
quantiles.

Objects of this class wil be typically created from an `FLQuant`. The
various statistics are calculated along the *iter* dimension of the
original `FLQuant` using `apply`.

</div>

<div class="section level2">

## Slots

-   .Data:

    The main array holding the computed statistics. `array`.

-   units:

    Units of measurement. `character`.

</div>

<div class="section level2">

## Accesors

-   mean,mean\<-::

    'mean' element on 6th dimension, arithmetic mean.

-   median,median\<-::

    'median' element on 6th dimension, median.

-   var,var\<-::

    'var' element on 6th dimension, variance.

-   lowq,lowq\<-::

    'lowq' element on 6th dimension, lower quantile (0.25 by default).

-   uppq,uppq\<-::

    'uppq' element on 6th dimension, upper quantile (0.75 by default).

-   quantile::

    returns the 'lowq' or 'uppq' iter, depending on the value of 'probs'
    (0.25 or 0.75).

</div>

<div class="section level2">

## Constructor

Inputs can be of class:

-   `FLQuant`::

    An FLQuant object with iters (i.e. dim6 \> 1)

</div>

<div class="section level2">

## Validity

-   iter::

    iter dimension is of length 5.

-   Dimnames::

    iter dimnames are 'mean', 'median', 'var', 'uppq' and'lowq'

</div>
