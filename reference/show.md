# Method show

Standard complete display of an object contents in an interactive
session. Objects of class [`FLArray`](FLArray.md) with length \> 1
alonArray sixth dimension (*iter*) are output in a summarised form, as
`median(mad)`, where mad is the median absolute deviation. See
[`mad`](https://rdrr.io/r/stats/mad.html).

## Usage

``` r
# S4 method for class 'FLArray'
show(object)

# S4 method for class 'FLArray'
print(x)
```

## Details

The same format is used for objects of class [`FLPar`](FLPar.md) with
length \> 1 on the last dimension (*iter*).

## Generic function

show(object)
