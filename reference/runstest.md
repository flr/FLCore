# Computes Runs Test p-values

Computes Runs Test p-values

## Usage

``` r
runstest(fit, obs, ...)

# S4 method for class 'FLQuants,missing'
runstest(fit, combine = TRUE)

# S4 method for class 'FLQuants,FLQuants'
runstest(fit, obs, combine = TRUE)

# S4 method for class 'FLQuant,FLQuant'
runstest(fit, obs, combine = TRUE)

# S4 method for class 'FLQuant,missing'
runstest(fit, combine = TRUE)

# S4 method for class 'numeric,numeric'
runstest(fit, obs, combine = TRUE)

# S4 method for class 'numeric,missing'
runstest(fit, obs, combine = TRUE)
```

## Arguments

- fit:

  The result of a model fit.

- obs:

  The observations used in the fit.

- ...:

  Extra arguments.

- combine:

  Should ages be combined by addition, defaults to TRUE.

## Value

A list with elements 'p.values' and 'pass'.
